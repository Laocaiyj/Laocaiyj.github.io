   在动手之前，需要先准备好一台目前还能正常使用的 Windows 主机；一块移动硬盘——实在找不到的话用 U 盘也行，但容量最好 64GB 起步，不然会显得呃...有些寒酸；一份 Linux 系统的 ISO 镜像文件，可以是 Ubuntu、Fedora 或 Debian，挑一个你看着顺眼的版本即可；还要提前装好 VirtualBox 以及对应的 Extension Pack，这两样都得有；最后，确保你的电脑支持 UEFI 启动，这个基本上 2020 年之后出厂的机器都能满足。

   在 Windows 里打开 磁盘管理，先深呼吸一下，别急。找到那块移动硬盘之后，逐个右键把上面的分区全部删干净。接着把分区表转换成 GPT，然后什么都别再做了，让它就那样乖乖地躺在“未分配”状态里——这一步很重要，手别贱，多点一下就容易把事情搞砸，真的会让人想骂一句该死。

   新建一个虚拟机，类型选 Linux——是的，这听起来就是废话。内存从 4GB 起步，硬盘这一步先别加。PS：VirtualBox 记得用管理员模式启动，不然到后面映射物理硬盘的时候会报错，然后你就得关进程、重开一遍，超蠢的，也很浪费时间。

映射整块移动硬盘

以管理员身份打开命令行，然后执行：

cd "C:\Program Files\Oracle\VirtualBox\"
VBoxManage internalcommands createrawvmdk ^
 -filename C:\usb.vmdk ^
 -rawdisk \\.\PhysicalDrive2

这里的 PhysicalDrive2 务必确认清楚是移动硬盘的盘号 选错盘原地起飞

   安装 Linux 的时候啊，眼睛瞪大点，目标磁盘必须选 /dev/sdb—— 也就是你那块被寄予厚望的移动硬盘，千万别手滑点到 /dev/sda，不然回头哭都找不着调。然后到了 EFI 分区这步，512MB 妥妥够用，格式乖乖选 FAT32，挂载到 /boot/efi。根分区 ext4，别去碰文件系统实验。
ps: 不会分区去网上搜

   装完之后设置好系统，拔掉硬盘，也就拥有一块移动Linux了

※ 本方法直接操作物理磁盘，需再三确认盘号
※ 适合需要“随身 Linux 环境”的人，比如你我这种折腾命
